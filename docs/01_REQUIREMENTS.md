# ResellerOS - Requirements

> This document defines what we're building.
> Updated in Phase 2: Planning based on Phase 1 Discovery.

## User Stories

### Listing Flow
- [ ] As a user, I can take photos and get a publish-ready draft in <30 seconds
- [ ] As a user, I answer at most 2 questions before draft is ready
- [ ] As a user, I can bulk-publish multiple drafts with one tap

### Autopilot
- [ ] As a user, offers are sent automatically with my floor protected
- [ ] As a user, stale listings are recovered without my intervention
- [ ] As a user, I can see what autopilot did and undo any action

### Cross-listing
- [ ] As a user, my items are available on eBay (native) from day one
- [ ] As a user, I can cross-list to Poshmark/Mercari (assisted) easily

### Profit Tracking
- [ ] As a user, I see true net per item (after fees, shipping, returns)
- [ ] As a user, I see weekly profit summary
